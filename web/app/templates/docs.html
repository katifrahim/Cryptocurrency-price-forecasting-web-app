User Documentation

Welcome to this page! Here we will guide you on how to navigate and use this website effectively!


REGISTERATION

Signup:

After coming to this website a new user would need to signup to this website by using the signup button that will be available in the navigation bar of the page in order to access the main features of the website.



Login:

After signing up to this page, the user would also need to login to this website by using the login button that will also be available in the navigation bar of the page in order to access the main features of the website.




PROFILE




After completing the registeration process, a profile button will appear instead of the signup and login button in the navigation bar of the page. This is the profile page of the user, where the user can view their user information and history. In the same page the user can edit their profile information, change their password and logout from the page.

Edit Profile:



After clicking on the edit profile button present in the profile page, a similar form like the signup form will appear where the user can change their information. After submitting this form the user would need to relogin into the page for security reasons.

Change Password:




After clicking on the change password button that is also present in the profile page, a password change form will appear where the user can change their password.

Logout:

The logout button will be available at the lower left side of the profile page. The user can only logout from the website using this button!


AI MODEL




After logging into the page the user will be able to access the AI model page where they can predict the future price of their favourite cryptocurrency with the help of Artificial Intelligence.

Input:

On the right hand side of the page, the user will have to input two values: "Cryptocurrency name" and "Number of days". The number of days refers to the number of days into the future of which the model will give the results. The cryptocurrency name refers to the cryptocurrency of which the model will give results. Currently, the model can predict upto 15 days into the future and supports three cryptocurrencies: Bitcoin, Ethereum and Dogecoin.

Output:

On the left hand side of the page, the results of the prediction will be displayed after the user has submitted the input values. The prediction graph is used to visualize the results of the prediction, while the prediction details showcases the trend and accuracy of the prediction. The accuracy of the prediction has been pre-defined after testing it multiple times with each input value.

Note:

The AI model page can take some time to show the results as it has to go through the entire training process in the backend. The duration of this process depends on the specification of the user's device.
